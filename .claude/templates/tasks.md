## タスク一覧

### [ ] Task 1: [型定義の作成]
- **担当**: Implementer
- **依存**: なし
- **ファイル**: src/types/[name].ts
- **見積もり**: 30分
- **完了条件**:
  - [ ] 型定義ファイルが作成されている
  - [ ] エクスポートされている
  - [ ] TSCでエラーが出ない

### [ ] Task 2: [テストフィクスチャの作成]
- **担当**: Implementer
- **依存**: なし
- **ファイル**: tests/fixtures/[name].ts
- **見積もり**: 30分
- **完了条件**:
  - [ ] テストデータが定義されている
  - [ ] 正常系・異常系のパターンがある

### [ ] Task 3: [APIハンドラの実装]
- **担当**: Implementer
- **依存**: Task 1, Task 2
- **ファイル**:
  - src/api/[name].ts
  - tests/api/[name].test.ts
- **見積もり**: 1時間
- **完了条件**:
  - [ ] テストが先に書かれている（TDD）
  - [ ] テストがすべてパスしている
  - [ ] 仕様書(docs/api_spec.yaml)と一致している

### [ ] Task 4: [統合テストの実装]
- **担当**: Implementer
- **依存**: Task 3
- **ファイル**: tests/integration/[name].test.ts
- **見積もり**: 45分
- **完了条件**:
  - [ ] E2Eシナリオのテストがある
  - [ ] エラーケースのテストがある
  - [ ] すべてのテストがパスしている

### [ ] Task 5: [ドキュメント更新]
- **担当**: TechLead
- **依存**: Task 4
- **ファイル**:
  - README.md
  - docs/[name].md
- **見積もり**: 30分
- **完了条件**:
  - [ ] 使用方法が記載されている
  - [ ] サンプルコードがある
  - [ ] 制約事項が明記されている
