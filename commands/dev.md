---

Invoke the fractal-dev-workflow:dev-workflow skill and follow it exactly as presented to you.

The user wants to start a development workflow. Use the skill to guide them through the 8 phases:
1. 質問 (Questioning)
2. 調査 (Investigation)
3. 契約設計 (Contract Design)
4. Codex計画レビュー (Codex Plan Review) - **Codex→User Approval Required**
5. 実装 (Implementation)
6. Codexコードレビュー (Codex Code Review) - **Codex→User Approval Required**
7. 検証 (Verification)
8. 運用設計 (Operations)
