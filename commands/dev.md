2. 調査 (Investigation)
3. 契約設計 (Contract Design)
4. Codex計画レビュー (Codex Plan Review) - **Codex自動レビュー（必須）**
5. 実装 (Implementation)
6. Chromeデバッグ (Chrome Debug)
7. Codexコードレビュー (Codex Code Review) - **Codex自動レビュー（必須）**
8. 検証 (Verification)
9. 運用設計 (Operations)
